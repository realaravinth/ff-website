ForgeFlux

API-space software forge federation for Gitea, Sourcehut, GitLab and GitHub

Imagine a world...
...where software development workflows are based on truly open standards.
 ...where you can choose who hosts your code while still interacting with the larger community.
 ...where freedom doesn't come at the cost of comfort and reliability.
 ...where Free Software developers are treated just as fairly as they treat their users.

Meet ForgeFlux

Where software forges are freed with external, API-space federation adapters.

Supported Software Forges

  • Gitea(WIP)
  • Sourcehut
  • GitLab
  • GitHub

ForgeFlux will implement external federation support for the following software forges: The project is highly modular and aims for maximum interoperability, so implementing support for other forges should be possible.
